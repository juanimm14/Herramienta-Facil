📊 Importar Índices de Aceptación

Subí el Excel con los índices de aceptación por slot. El archivo debe tener el formato con columnas: SLOT, Isla, MARCA, Modelo, Validador, ENERO, FEBRERO... DICIEMBRE (los porcentajes de cada mes).

📂
Arrastrá el Excel aquí o hacé click para seleccionar
Formatos aceptados: .xlsx, .xls
📋 Formato esperado del Excel
• Col. A: SLOT  •  Col. B: Isla  •  Col. C: MARCA  •  Col. D: Modelo  •  Col. E: Validador
• Col. F–Q: ENERO, FEBRERO, MARZO... DICIEMBRE (porcentaje de aceptación, ej: 89.27%)
• Cada fila = un slot. Los promedios generales deben estar en una fila separada con "PROMEDIOS MENSUAL" o similar.
